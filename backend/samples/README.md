# Sample test images

For ethical POC testing, source test IDs from one of:

- **Aadhaar**: UIDAI's published sample / specimen cards (downloadable from uidai.gov.in). Many YOLO-Aadhaar repos include sample images too.
- **PAN**: your own PAN with consent, or generic dummy PAN images from training datasets.
- **Driving Licence / Passport / Voter ID**: your own with consent, or open synthetic datasets.

Place test images here (this directory is gitignored). Keep filenames descriptive (`aadhaar-clean.jpg`, `pan-rotated.jpg`, etc.) so it's easy to A/B different inputs.
